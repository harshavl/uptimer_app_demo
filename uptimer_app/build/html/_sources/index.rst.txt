Welcome to Uptimer App's documentation!
=======================================

Uptimer is a CLI tool to monitor the status of websites.
Specify a URL, and it will return its HTTP status code.

Quickstart
----------
1. Install all requirements:
   $ make install
   
2. Run Uptimer
   $ make test
